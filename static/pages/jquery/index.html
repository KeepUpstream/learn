2.基本语法

$(selector).action() 所以，接下来首先要学的就是selector选择器和action可选操作。 $(document).ready(function(){ // 开始写 jQuery 代码... }); 简写为：$(function(){}); jQuery 的入口函数是在 html 所有标签(DOM)都加载之后，就会去执行。 可执行n次，第N次的执行不会被last那次所覆盖。 JavaScript 的 window.onload 事件是等到所有内容，包括外部图片之类的文件加载完后，才会执行。 只能执行1次，会覆盖。
